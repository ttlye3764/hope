Customization

1. How to add new page?

We have provided a starter page (check src/html/pages-starter.html) which allows you to get started easily and helps you to add new page. Please note following some important points.

  • Make sure you have included css/app.min.css, css/icons.min.css and js/app.min.js in your html. If you are using dark, green or custom created theme, make sure to include the css/app-dark.min.css, or css/app-green.min.css or custom theme css file.
  • Most of default/basic form elements along with few advanced elements are available and bundled in above css and js and so you don't need to include any css or js separately.
  • Few elements e.g. charts, data tables, calendar, maps etc requires you to include corresponding css and js files in your html. Please check corresponding documentation page for the same.

2. How to change color palette or add new theme?

Hyper comes with three default themes. I.e. Default (Purple), Dark and Green. However you can either modify the existing theme and the colors of it or add completely new theme.

In order to modify the colors in existing themes, open the corresponding /_variables.scssfile from src/scss/config/<THEME_NAME> and you can change any values in it. Your changes would get reflected automatically in any bootstrap based components or elements.

If you wish to create new theme, just copy the folder src/scss/config/default and rename it. After that just make the color or other changes in it's _variables.scss file. Also copy src/scss/app.scss and change the import lines from default theme to your newly created theme folder. E.g. Replace following:

                                                        
                                                            @import "config/default/variables";
                                                            @import "config/default/custom-variables";
With
                                                        
                                                            @import "config/YOUR_THEME/variables";
                                                            @import "config/YOUR_THEME/custom-variables";

After this change, modify gulpfile.js - function css() to compile the custom theme scss file.

3. How to change style of a particular UI element?

In order to change the particular UI element, widget, chart, etc, please check folder src/scss/custom/plugins. You should be able to change any style by simply making modifications in corresponding file here.
