File Uploads

Easily file input in your form with drag and drop support and with nice file previews.

Make sure to include following js files at end of body element

File Upload Example

Just specify the data attribute data-plugin='dropzone' to have file upload widget with drag and drop support in your form element.

By default the file previews would be generated within same element. If you would like to have then generated in different place (like in the example below), just specify the data attribute data-previews-container="".

Also you can easily provide the custom template to show preview item with customize look and feel. Just create a html template for it and specify it as value of an attribute data-upload-preview-template="".
