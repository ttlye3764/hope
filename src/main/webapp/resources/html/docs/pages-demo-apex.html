Charts using Apex Charts

ApexCharts is an open-source modern charting library that helps developers to create interactive visualizations for web pages.

Following are simple examples of line and bar charts, but you can create many different types of charts very easily. You can refer to official documentation here. Also checkout js file demo.apexchart.* files for many more examples of all types of charts.
