Form Advanced Components

Select 2

Select2 is a replacement for select boxes. It supports searching, remote data sets, and infinite scrolling of results. In order to replace a regular select element with select2 simply provide an attribute called data-toggle="select2". You can specify other options supported by select2 as well. Please refer to documentation here

How to customize the style of select2?

Open file src/scss/custom/plugins/_select2.scss and change the different style to change the background, text or other colors.

Date Range Picker

A JavaScript component for choosing date ranges, dates and times. In order to enable support of date picker on a regular input element, simply provide an attribute called data-toggle="data-picker". You can specify other options supported by datapciker as data attribute as well. Please refer to documentation here

How to customize the style of date-range-picker?

Open file src/scss/custom/plugins/_daterange.scss and change the different style to change the background, text or other colors.

Switch

Switches can be used as replacement of default checkbox element. The swicthes are pure css based and can replace a regular checkbox into beautiful switch by simply specifying an attribute data-switch="".

How to customize the style of switch?

Open file src/scss/custom/components/_switch.scss and change the different style to change the background, text or other colors.

Bootstrap Touchspin

A mobile and touch friendly input spinner component for Bootstrap. Specify attribute data-toggle="touchspin" and your input would be conveterted into touch friendly spinner. You can specify all options as data attributes. More options can be found in official documentation here.

Bootstrap Maxlength

Uses the HTML5 attribute "maxlength" to work. Just specify data-toggle="maxlength" attribute to have maxlength indication on any input. You can specify all options as data attributes. More options can be found in official documentation here.

The badge will show up by default when the remaining chars are 10 or less:

Satrt displaying the indication when reached to some threshhold. Specift the data attribute threshold. E.g. data-threshold="12"

All you need to do is specify the data attribute placement. The possible positions are left, top, right, bottom-right, top-right, top-left, bottom, bottom-left and centered-right. If none is specified, the positioning will be defauted to 'bottom'. E.g. data-placement="top"

Bootstrap maxlength supports textarea as well as inputs. Even on old IE.

Timepicker

Easily select a time for a text input using your mouse or keyboards arrow keys. Specify attribute data-toggle="timepicker" and you would have nice timepicker input element. You can specify all options as data attributes. More options can be found in official documentation here.

Typeahead

A flexible JavaScript library that provides a strong foundation for building robust typeaheads. More options can be found in official documentation here.

Bootstrap-datepicker provides a flexible datepicker widget in the Bootstrap style.
