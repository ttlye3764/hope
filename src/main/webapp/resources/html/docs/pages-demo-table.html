Table

Tables are being used widespread and are kind of mandatory for most of web applications.

How to customize the style of tables?

Open file src/scss/custom/components/_tables.scss and change the different style to change the background, text or other colors.

Basic example

For basic styling—light padding and only horizontal dividers—add the base class .table to any <table>.
