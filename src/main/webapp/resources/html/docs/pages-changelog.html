Change Log

v2.0.0 - 21 May 2019
  • Added: Webpack based version
  • Improved: Layout support and added javascript api to manage the layout and other customization
  • Updated: Gulp 4 based build flow
  • Improved: Left side menu - upto 3 level support
  • Added: New Green color based theme
  • Added: Email application pages
  • Added: Dark left sidebar style
  • Added: Preloader
  • Improved: More data tables examples
  • Fixed Bug - When dropdown is at bottom of page, the dropdown items were not being displayed
  • Updated: Metismenu, Jquery, chartjs, apexcharts, britecharts, etc to latest version
v1.5.0 - 15 Feb 2019
  • Updated: Bootstrap to latest (v4.3.1)
  • Added new layout - having detached sidebar
  • Added new dashboard for Project management
  • Improved left sidebar layout. You can have now scrollable sidenav option
  • Added Bootstrap Datepicker
  • Added new radar chart from apex
  • Multiple plugins (Apexchats, Datatables, Moment, Popper.js, Britecharts, Chartjs, Material Design Icons, etc) are upgraded to latest version
v1.4.0 - 10 Jan 2019
  • Upgraded to latest bootstrap (v4.2.1)
  • Added new dark color based layout
  • Added new dashboard for CRM usecase
  • Added new page to demo spinners from latest bootstrap
  • Added demo for bootstrap based toast
  • Added multiple new widgets including chat, todo, statistical, etc
  • Added sparkline chart support
  • Added typeahead input control
  • Added a dropdown selection in navbar to allow selection of language
  • Improved scss to control the height of top navbar using variables
  • Upgraded apex chart, material design, etc to latest version
  • Bug Fix - Topbar shows the logo in horizontal view in small screen
v1.3.0 - 25 Sep 2018
  • Added new ecommerce app related pages including product listing, product detail, order list, order detail, shopping cart, checkout, seller listing, etc
  • Add new widgets including having beutiful charts in it
  • Add boxed (fixed width) layouts having verticle as well as horizontal menu support
  • Changed icons of datatables to have better visibility around filtering
  • Added new apex chart called "candlestick" (most favorite chart of crypto trade platform)
  • Fixed issue with scroll in right sidebar
  • Upgraded apex chart version
v1.2.0 - 01 Sep 2018
  • Added more components including timepicker, spinner, max length, WYSIWYG editors, etc.
  • Added beautiful apex charts having many more examples and variations of charts
  • Added more variations of modals, alerts
  • Added variations of authentication related pages (login, signup, recover password, etc)
v1.1.0 - 07 Aug 2018
  • Upgraded to Bootstrap v4.1.3
  • Added multiple layout options including support for horizontal nav, collapsed (mini), light sidenav, etc
  • Added more sample apps and pages including project list page, project detail page, task board, task detail dialog, etc
  • Added new components including dragula, file upload, etc
  • Improved form control input style
  • Separated font icons related styles from main style file (app.css)
  • Improved the top right menu item. Also changed the form input element style
  • Added more examples of google maps (with custom styles)
